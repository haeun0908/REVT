# 수용 어휘 검사 웹 애플리케이션(수정중)

본 프로그램은 기존에 수기로 진행하던 수용 어휘 검사 과정에서 발생하는 번거로움을 해소하고자, 나사렛대학교 언어치료학과 학생들의 의견을 반영하여 개발한 Flask 기반 웹 애플리케이션이다. 사용자는 온라인 환경에서 손쉽게 검사를 진행할 수 있으며, 결과는 화면과 PDF 파일로 바로 확인할 수 있다. 또한 연령에 따라 검사 시작 문항을 자동으로 지정하고, 기초선과 최고한계선을 기준으로 검사 종료 시점을 자동으로 판정하여 전반적인 과정을 효율적으로 자동화하였다.

## Ⅱ. 실행 방법

본 프로그램은 Python을 별도로 설치하지 않아도 바로 실행할 수 있도록 실행 파일(.exe) 형태로 제공된다. 사용자는 별도의 개발 환경 설정이나 복잡한 명령어 입력 없이도 프로그램을 쉽게 구동할 수 있다.

프로그램의 사용 방법은 다음과 같다.

2-1. 제공된 ZIP 파일을 먼저 다운로드한다.<br>
![이미지 설명](이미지_경로)

2-2. 다운로드한 ZIP 파일의 압축을 해제한다.<br>
![이미지 설명](이미지_경로)

2-3. 압축 해제 후 생성된 폴더 안에서 아래의 경로를 따라 이동한다.
```
\dist\app.exe
```
![이미지 설명](이미지_경로)

2-4. app.exe 파일을 더블 클릭하여 실행한다.<br>
![이미지 설명](이미지_경로)

2-5. 잠시 기다리면 웹 브라우저가 자동으로 열리며, 프로그램 화면이 표시된다.<br>
![이미지 설명](이미지_경로)

## Ⅲ. 검사 절차

3-1. 연습문항:<br>
- 검사자가 6세 미만인 경우: A, B, C / 6세 이상인 경우: D, E<br>
- 연습문항에서 대상자가 틀리게 반응하면 정확한 답을 알려준 후 아동이 맞는 반응을 할때까지 그 문항을 반복<br>
- 대상자가 검사자의 도움없이 연습문항을 모두 정확하게 반응하면 본 문항을 실시<br>
- 많은 연습을 계속했는데도 아동이 적절하게 반응하지 못하면 검사를 중지

※ 본 프로젝트에 사용된 문항 데이터는 나사렛대학교 언어치료학과 학생에게 제공받아 구성되었다.
   
3-2. 검사실시:<br>
- 연습문항이 끝나면, 대상자의 생활연령에 따른 시작문항을 참고하여 본 검사문항을 실시

|연령|시작문항 번호|
|---|---|
|2~4세|1|
|5~6세|16|
|7~9세|46|
|10~12세|91|
|13세 이상|121|
   
3-3. 한계선<br>
- 검사는 최고한계선문항까지 실시하는 것을 원칙으로 하며 최고한계선이 확정되지 않은 경우는 끝 문장까지 실시한다. 기초선이 확립되면 계속 윗번호의 어려운 문항들을 검사해 나가다가 대상자가 연속적으로 8개 문항 중 6개를 틀리게 반응하면 검사를 중지한다. 이 때 틀리게 반응한 마지막 문항을 최고한계선이라고 한다. 최고한계선이 정해지면 그보다 높은 번호의 문항들은 검사를 실시하지 않아도 틀린 것으로 간주한다.

||방법|
|---|---|
|기초선|연속해서 8개 문항을 맞추는 경우|
|최고한계선|연속된 8개 문항중에서 6개를 틀리는 경우|

## 4. 문항 구성

* **연습 문항**: A, B, C, D, E
* **본 검사 문항**: 1번 ~ 185번

각 문항은 다음 정보로 구성된다.

```text
(문항번호, 문항 내용, 정답 번호)
```

모든 문항 데이터는 `ALL_ITEMS` 리스트에 저장되어 있다.

※ 본 프로젝트에 사용된 문항 데이터는 **나사렛대학교 언어치료학과 학생의 협조로 제공**받아 구성되었다.

## 6. 검사 진행 방식

### 6.1 연습 문항

* 6세 미만: A, B, C
* 6세 이상: D, E
* 연습 문항을 **모두 정답(+)**일 경우 본 검사 진행
* 하나라도 오답(-)일 경우 검사 종료

---

### 6.2 본 검사

* 각 문항에 대해 `+`(정답) 또는 `-`(오답) 입력
* 응답은 세션(session)에 순서대로 저장

#### 기초선(Base Line)

* 최근 8문항이 모두 정답(+)인 경우
* 최초로 조건을 만족한 문항을 기초선으로 설정

#### 최고한계선(Ceiling)

* 최근 8문항 중 6문항 이상 오답(-)인 경우
* 검사 자동 종료
