<h3 align="center">요 약</h3>
본 프로그램은 기존에 수기로 진행하던 수용 어휘 검사 과정에서 발생하는 번거로움을 해소하고자, 나사렛대학교 언어치료학과 학생들의 의견을 반영하여 개발한 Flask 기반 웹 애플리케이션이다. 사용자는 온라인 환경에서 손쉽게 검사를 진행할 수 있으며, 결과는 화면과 PDF 파일로 바로 확인할 수 있다. 또한 연령에 따라 검사 시작 문항을 자동으로 지정하고, 기초선과 최고한계선을 기준으로 검사 종료 시점을 자동으로 판정하여 전반적인 과정을 효율적으로 자동화하였다.

<h3 align="center">Ⅰ. 프로그램 구조</h3>

```
REVT/
│
├─ app.py              # 메인 프로그램
│                      
│
├─ dist/
│  └─ app.exe          # 실행 파일
│                    
│
├─ static/             # 디자인 및 리소스
│                      
│
└─ templates/          # 웹 화면 구성
```
본 프로그램은 별도의 개발 환경 설정이나 복잡한 명령어 입력 없이 바로 실행할 수 있도록 실행 파일(.exe) 형태로 제공되며, 저작권 보호를 위해 소스 코드는 공개하지 않는다.

<h3 align="center">Ⅱ. 검사 절차</h3>

2-1. 연습 문항
- 검사자가 6세 미만일 경우에는 A, B, C 문항을, 6세 이상일 경우에는 D, E 문항을 연습 문항으로 제시한다[1].
- 연습 문항을 모두 정답(+)으로 응답한 경우에만 본 검사를 진행한다.
- 연습 문항 중 하나라도 오답(-)이 있을 경우 검사를 종료한다.

※ 본 프로그램에 사용된 문항 데이터는 나사렛대학교 언어치료학과 학생에게 제공받아 구성되었다.

2-2. 검사 실시
- 연습 문항까지 마친 뒤, 대상자의 생활연령에 따라 아래의 [표 1]에 제시된 시작 문항 번호를 참고하여 본 검사를 실시한다[1].
- 각 문항의 반응은 정답은 '+', 오답은 '-'로 기록한다.
- 응답은 검사 세션에 순차적으로 저장한다.

표 1. 연령별 시작 문항
|연령|시작 문항 번호|
|---|---|
|2~4세|1|
|5~6세|16|
|7~9세|46|
|10~12세|91|
|13세 이상|121|

2-3. 한계선
- 검사 진행 시, 최고한계선 문항까지 실시하는 것을 원칙으로 한다.
- 만약 최고한계선이 확정되지 않았다면 마지막 문항까지 검사를 계속한다.
- 기초선이 확립되면, 윗 번호의 더욱 난이도가 높은 문항들을 추가로 진행한다.
- 이때, 검사자가 연속된 8개 문항 중 6개에 오답으로 반응하면 검사를 중단한다.
- 연속해서 6개를 오답 처리한 마지막 문항을 최고한계선으로 간주한다.
- 최고한계선이 정해지면, 그 이후의 모든 문항은 별도로 실시하지 않아도 오답으로 처리한다[1].

표 2. 기초선과 최고한계선 설정방법
||방법|
|---|---|
|기초선|연속해서 8개 문항을 맞추는 경우|
|최고한계선|연속된 8개 문항 중에서 6개를 틀리는 경우|

<h3 align="center">Ⅲ. 결과 화면</h3>

3-1. 검사 완료 안내
- 사용자가 검사를 마치면, 검사 완료 메시지가 화면에 안내된다.

3-2. PDF 결과지 다운로드
- 결과 리포트 저장하기 버튼을 누르면 PDF 파일로 결과를 내려받을 수 있다.

3-3. 검사 결과 요약
- 검사 결과 요약 영역에는 기초선, 최고한계선, 오답 문항 목록이 순차적으로 포함되어 있다.
- 기초선과 최고한계선은 각각 해당 문항 번호와 연관된 단어를 함께 표기한다.
- 오답 문항 목록에는 틀린 문항의 번호와 해당 단어가 정리된다.

3-4. 전체 문항 결과 안내
- 전체 문항 결과는 PDF에서도 확인할 수 있으며, 각 문항의 번호, 단어, 응답 결과(정답은 +, 오답은 -로 표기)가 함께 표시된다.
- 시작 문항, 기초선, 최고한계선은 별도의 표시로 강조한다.

3-5. 연령별 시작 문항 안내
- 사용자의 연령대에 따라 시작 문항이 다르기 때문에, 관련 안내가 화면에 표시된다.

3-6. 재검사
- 검사하고자 할 때는 '다시 검사하기' 버튼을 눌러 검사를 재진행할 수 있다.
