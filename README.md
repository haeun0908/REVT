# 수용 어휘 검사 웹 애플리케이션

본 프로그램은 기존에 수기로 진행하던 수용 어휘 검사 과정에서 발생하는 번거로움을 해소하고자, 나사렛대학교 언어치료학과 학생들의 의견을 반영하여 개발한 Flask 기반 웹 애플리케이션이다. 사용자는 온라인 환경에서 손쉽게 검사를 진행할 수 있으며, 결과는 화면과 PDF 파일로 바로 확인할 수 있다. 또한 연령에 따라 검사 시작 문항을 자동으로 지정하고, 기초선과 최고한계선을 기준으로 검사 종료 시점을 자동으로 판정하여 전반적인 과정을 효율적으로 자동화하였다.

## 0. 실행 방법

본 프로그램은 Python을 별도로 설치하지 않아도 바로 실행할 수 있도록 실행 파일(.exe) 형태로 제공된다. 사용자는 별도의 개발 환경 설정이나 복잡한 명령어 입력 없이도 프로그램을 쉽게 구동할 수 있다.

프로그램의 사용 방법은 다음과 같다.

① 제공된 ZIP 파일을 먼저 다운로드한다.<br>
![이미지 설명](이미지_경로)<br>
② 다운로드한 ZIP 파일의 압축을 해제한다.<br>
![이미지 설명](이미지_경로)<br>
③ 압축 해제 후 생성된 폴더 안에서 아래의 경로를 따라 이동한다.
```
\dist\app.exe
```
![이미지 설명](이미지_경로)<br>
④ app.exe 파일을 더블 클릭하여 실행한다.<br>
![이미지 설명](이미지_경로)<br>
⑤ 잠시 기다리면 웹 브라우저가 자동으로 열리며, 프로그램 화면이 표시된다.<br>
![이미지 설명](이미지_경로)

## 1. 주요 기능

* 연령 입력 기반 검사 시작 문항 자동 설정
* 연습 문항 → 본 검사 단계적 진행
* 기초선(Base Line) 자동 판정
* 최고한계선(Ceiling) 도달 시 검사 자동 종료
* 오답 문항 요약 결과 제공
* 검사 결과 PDF 파일 다운로드
* 검사 중 되돌리기(Undo) 및 초기화 기능

## 4. 문항 구성

* **연습 문항**: A, B, C, D, E
* **본 검사 문항**: 1번 ~ 185번

각 문항은 다음 정보로 구성된다.

```text
(문항번호, 문항 내용, 정답 번호)
```

모든 문항 데이터는 `ALL_ITEMS` 리스트에 저장되어 있다.

※ 본 프로젝트에 사용된 문항 데이터는 **나사렛대학교 언어치료학과 학생의 협조로 제공**받아 구성되었다.

| 연령     | 시작 문항 |
| ------ | ----- |
| 2~4세   | 1번    |
| 5~6세   | 16번   |
| 7~9세   | 46번   |
| 10~12세 | 91번   |
| 13세 이상 | 121번  |

## 6. 검사 진행 방식

### 6.1 연습 문항

* 6세 미만: A, B, C
* 6세 이상: D, E
* 연습 문항을 **모두 정답(+)**일 경우 본 검사 진행
* 하나라도 오답(-)일 경우 검사 종료

---

### 6.2 본 검사

* 각 문항에 대해 `+`(정답) 또는 `-`(오답) 입력
* 응답은 세션(session)에 순서대로 저장

#### 기초선(Base Line)

* 최근 8문항이 모두 정답(+)인 경우
* 최초로 조건을 만족한 문항을 기초선으로 설정

#### 최고한계선(Ceiling)

* 최근 8문항 중 6문항 이상 오답(-)인 경우
* 검사 자동 종료

---

## 7. 결과 화면

결과 페이지에서는 다음 정보를 제공한다.

* 오답 문항 목록 (문항 번호 + 단어)
* 기초선 문항 번호 및 단어
* 최고한계선(마지막 응답 문항) 정보

---

## 8. PDF 결과지 다운로드

* ReportLab을 사용하여 PDF 생성
* Malgun Gothic 글꼴 사용
* 포함 내용

  * 오답 문항 요약
  * 전체 문항 표
  * 시작 문항 / 기초선 / 최고한계선 표시
  * 연령별 시작 문항 안내 문구

PDF 파일명: `검사결과지.pdf`

---

## 9. 기타 기능

* **Undo**: 마지막 문항 응답 취소
* **Reset**: 검사 초기화 후 처음 화면으로 이동

## 11. 프로젝트 목적

본 프로젝트는 연령 기반 수용 어휘 검사를 **웹 환경에서 체계적으로 자동화**하고, 검사 결과를 **객관적으로 기록 및 문서화(PDF)**하는 것을 목표로 한다.

---

## 12. 참고 사항

* PDF 출력 시 `static/fonts/malgun.ttf` 파일이 반드시 필요하다.
* 세션 기반 구조이므로 브라우저 종료 시 검사 정보가 초기화될 수 있다.
