# 수용 어휘 검사 웹 애플리케이션(수정중)

본 프로그램은 기존에 수기로 진행하던 수용 어휘 검사 과정에서 발생하는 번거로움을 해소하고자, 나사렛대학교 언어치료학과 학생들의 의견을 반영하여 개발한 Flask 기반 웹 애플리케이션이다. 사용자는 온라인 환경에서 손쉽게 검사를 진행할 수 있으며, 결과는 화면과 PDF 파일로 바로 확인할 수 있다. 또한 연령에 따라 검사 시작 문항을 자동으로 지정하고, 기초선과 최고한계선을 기준으로 검사 종료 시점을 자동으로 판정하여 전반적인 과정을 효율적으로 자동화하였다.

## Ⅰ. 프로그램 구조

```
REVT/
│
├─ app.py              # 프로그램 핵심 파일
│                      
│
├─ dist/
│  └─ app.exe          # 프로그램 실행 파일
│                    
│
├─ static/             # 화면 디자인과 PDF 출력에 사용되는 파일
│                      
│
└─ templates/          # 프로그램 화면(시작·진행·결과)을 구성하는 파일
```

## Ⅱ. 실행 방법

본 프로그램은 Python을 별도로 설치하지 않아도 바로 실행할 수 있도록 실행 파일(.exe) 형태로 제공된다. 사용자는 별도의 개발 환경 설정이나 복잡한 명령어 입력 없이도 프로그램을 쉽게 구동할 수 있다.

프로그램의 사용 방법은 다음과 같다.

2-1. 제공된 ZIP 파일을 먼저 다운로드한다.<br>
![이미지 설명](이미지_경로)

2-2. 다운로드한 ZIP 파일의 압축을 해제한다.<br>
![이미지 설명](이미지_경로)

2-3. 압축 해제 후 생성된 폴더 안에서 아래의 경로를 따라 이동한다.
```
\dist\app.exe
```
![이미지 설명](이미지_경로)

2-4. app.exe 파일을 더블 클릭하여 실행한다.<br>
![이미지 설명](이미지_경로)

2-5. 잠시 기다리면 웹 브라우저가 자동으로 열리며, 프로그램 화면이 표시된다.<br>
![이미지 설명](이미지_경로)

## Ⅲ. 검사 절차

3-1. 연습문항:<br>
- 검사자가 6세 미만일 경우에는 A, B, C 문항을, 6세 이상일 경우에는 D, E 문항을 연습문항으로 제시한다[1].
- 연습문항을 모두 정답(+)으로 응답한 경우에만 본 검사를 진행한다.
- 연습문항 중 하나라도 오답(-)이 있을 경우 검사를 종료한다.

※ 본 프로젝트에 사용된 문항 데이터는 나사렛대학교 언어치료학과 학생에게 제공받아 구성되었다.
   
3-2. 검사실시:<br>
- 연습문항까지 마친 뒤, 대상자의 생활연령에 따라 아래의 표에 제시된 시작문항 번호를 참고하여 본 검사를 실시한다[1].
- 각 문항의 반응은 정답은 '+', 오답은 '-'로 기록한다.
- 응답은 검사 세션에 순차적으로 저장한다.

|연령|시작문항 번호|
|---|---|
|2~4세|1|
|5~6세|16|
|7~9세|46|
|10~12세|91|
|13세 이상|121|
   
3-3. 한계선<br>
- 검사 진행 시, 최고한계선 문항까지 실시하는 것을 원칙으로 한다.
- 만약 최고한계선이 확정되지 않았다면 마지막 문항까지 검사를 계속한다.
- 기초선이 확립되면, 윗 번호의 더욱 난이도가 높은 문항들을 추가로 진행한다.
- 이때, 검사자가 연속된 8개 문항 중 6개에 오답으로 반응하면 검사를 중단한다.
- 연속해서 6개를 오답 처리한 마지막 문항을 최고한계선으로 간주한다.
- 최고한계선이 정해지면, 그 이후의 모든 문항은 별도로 실시하지 않아도 오답으로 처리한다[1].

||방법|
|---|---|
|기초선|연속해서 8개 문항을 맞추는 경우|
|최고한계선|연속된 8개 문항중에서 6개를 틀리는 경우|
